# spotpull v1.3.0
Spotpull é uma ferramenta para baixar arquivos, incluindo músicas, do Spotify

---

# Funcionalidades
- (Novo!) Baixar músicas 
- Baixar capas de albuns
- Baixar imagens de playlists
- Baixar foto de perfil de artistas
- Baixar foto de perfil de usuários

---

## Instalação

```
git clone https://github.com/mvrheitor/spotpull.git
cd spotpull
pip install -r requirements.txt
cp .env.example .env
```
Coloque seu client ID e secret do Spotify no .env.

## Uso
```
python run.py
```

## Requisitos:
- API do Spotify
