# spotpull v1.0.0
Spotpull é uma ferramenta para baixar imagens do Spotify (capas de albuns e playlists)

---

# Funcionalidades
- Baixar capas de albuns a partir do link do album
- Baixar imagnes de playlists a partir do link da playlist

---

## Instalação

```
git clone https://github.com/mvrheitor/spotpull.git
cd spotpull
pip install -r requirements.txt
cp .env.example .env
```
Coloque seu client ID e secret do Spotify no .env.

## Uso
```
python run.py
```

## Requisitos:
- API do Spotify
